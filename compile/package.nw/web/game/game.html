Changelog

v1.0 - Initial Release

  • Official release of Granny Clicker!
  • Core Granny clicker mechanics with a dynamic multiplier and unique scale animations.
  • Upgrade store featuring Double Coins (2x multiplier) and Auto Clicker.
  • Threat system: Granny randomly spawns counting down to an immersive fullscreen jumpscare.
  • Advanced audio mixer with independent volume sliders and live percentage indicators.
  • Selectable sound pack styles (Classic Granny and Project MITAI).
  • Full localization for 3 languages: English, Japanese, and Polish (alphabetically sorted).
  • Atmospheric audio & visuals: randomized location backgrounds, looping house music, ambient sounds, and original Granny voice lines.
  • Built-in anti-cheat system detecting suspiciously fast clicks with atmospheric resets.
  • Custom Granny mouse cursor optimized for PC players.
